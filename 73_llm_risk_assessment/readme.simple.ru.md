# Оценка рисков с помощью LLM — простым языком

> Объяснение для начинающих и студентов, знакомящихся с машинным обучением

## Что такое риск в трейдинге?

Представь, что ты собираешься перейти оживлённую дорогу. Перед тем как шагнуть с бордюра, ты естественно оглядываешься:

- Едут ли машины?
- Как быстро они движутся?
- Горит ли зелёный свет?

Это **оценка риска** — проверка опасностей перед действием.

В трейдинге мы делаем то же самое перед покупкой или продажей:

- Здорова ли компания?
- Ожидаются ли плохие новости?
- Каково общее настроение рынка?

## Что такое LLM?

**LLM** означает **Large Language Model** (Большая Языковая Модель). Это ИИ, который научился читать и понимать текст, прочитав миллиарды документов — книги, сайты, новостные статьи и многое другое.

Думай об этом как об очень начитанном друге, который:
- Прочитал все газеты за последние 10 лет
- Помнит финансовые отчёты тысяч компаний
- Может быстро резюмировать длинные документы
- Понимает контекст и нюансы в языке

### Популярные LLM

| Название | Компания | Что делает |
|----------|----------|------------|
| ChatGPT | OpenAI | Общение и анализ |
| Claude | Anthropic | Вдумчивый анализ и код |
| Gemini | Google | Поиск и рассуждение |
| FinGPT | Open source | Специализация на финансах |

## Как LLM оценивают риски?

### Старый способ (Вручную)

```
Человек-аналитик читает → 100 страниц финансовых отчётов
        ↓
Занимает → 3 часа
        ↓
Пишет → 1 оценку риска
```

### Новый способ (С LLM)

```
LLM читает → 100 страниц финансовых отчётов
        ↓
Занимает → 30 секунд
        ↓
Пишет → 1 оценку риска
```

LLM может делать это для сотен компаний одновременно!

## Типы рисков, которые ищет LLM

### 1. Рыночный риск
**Что это:** Весь рынок может упасть и утянуть твои инвестиции за собой.

**Пример из жизни:** Во время COVID-19 в марте 2020 почти все акции упали, даже хороших компаний.

**Что ищет LLM:**
- Новости о рецессии
- Объявления центрального банка
- Упоминания глобального кризиса

### 2. Кредитный риск
**Что это:** Компания может не выплатить свои долги.

**Пример из жизни:** Если компания занимает слишком много денег и не может вернуть, она может обанкротиться.

**Что ищет LLM:**
- Упоминания уровня долга
- Изменения кредитного рейтинга
- Новости о пропущенных платежах

### 3. Риск ликвидности
**Что это:** Ты можешь не успеть продать, когда захочешь.

**Пример из жизни:** Представь, что у тебя акции крошечной компании. Когда захочешь продать, покупателей может не быть!

**Что ищет LLM:**
- Упоминания низкого торгового объёма
- Опасения делистинга
- Язык о заморозке рынка

### 4. Операционный риск
**Что это:** Что-то может пойти не так внутри компании.

**Пример из жизни:** CEO внезапно увольняется, или на заводе случается пожар.

**Что ищет LLM:**
- Смена руководства
- Судебные иски
- Технические сбои
- Скандалы

### 5. Регуляторный риск
**Что это:** Правительство может создать новые правила, которые навредят компании.

**Пример из жизни:** Новые экологические законы могут сильно усложнить бизнес угольной компании.

**Что ищет LLM:**
- Упоминания нового законодательства
- Штрафы и санкции
- Правительственные расследования

### 6. Сентимент-риск
**Что это:** Отношение людей к компании может измениться.

**Пример из жизни:** Вирусное видео показывает плохое обслуживание клиентов, и все перестают покупать у этой компании.

**Что ищет LLM:**
- Тон социальных сетей
- Настроение заголовков новостей
- Сдвиги общественного мнения

## Как работает оценка риска

LLM даёт каждому риску оценку от 1 до 10:

```
Оценка │  Значение                    │  Действие
───────┼──────────────────────────────┼────────────────
1-2    │  Очень безопасно             │  Можно инвестировать больше
3-4    │  Низкий риск                 │  Обычная позиция
5-6    │  Средний риск                │  Быть осторожным
7-8    │  Высокий риск                │  Уменьшить позицию
9-10   │  Зона опасности              │  Подумать о выходе
```

### Пример отчёта о рисках

```
Компания: Example Corp (EXMP)
Дата: Сегодня

Оценка рисков:
├── Рыночный риск:     4/10 (Низкий)
├── Кредитный риск:    3/10 (Низкий)
├── Риск ликвидности:  2/10 (Очень низкий)
├── Операционный риск: 7/10 (Высокий) ← Беспокойство!
├── Регуляторный риск: 5/10 (Средний)
└── Сентимент-риск:    6/10 (Средний)

Общая оценка: 4.5/10 (Умеренный)

Ключевой вывод: CEO объявил о неожиданном уходе.
                Новое руководство несёт неопределённость.

Рекомендация: Уменьшить позицию на 25% до
              назначения нового CEO.
```

## Принятие торговых решений

### Простые правила решений

```python
# Это псевдокод (не настоящий код, просто показ логики)

if risk_score <= 4:
    action = "ПОКУПАТЬ больше"
    reason = "Низкий риск, хорошая возможность"

elif risk_score <= 6:
    action = "ДЕРЖАТЬ"
    reason = "Умеренный риск, ждём и смотрим"

elif risk_score <= 8:
    action = "УМЕНЬШИТЬ"
    reason = "Высокий риск, защити свои деньги"

else:
    action = "ВЫХОД"
    reason = "Слишком рискованно, уходи!"
```

### Размер позиции

Сумма, которую ты инвестируешь, должна соответствовать риску:

```
Низкий риск    → Большая позиция   (инвестировать больше)
      ↑
      │
      │
      ↓
Высокий риск   → Меньшая позиция   (инвестировать меньше)
```

## Простой пример

Давай пройдём полный пример:

### Шаг 1: Получить новости

```
ЗАГОЛОВОК: "Компания по майнингу Bitcoin сообщает о сильной прибыли"

СТАТЬЯ: "CryptoMine Inc. объявила о доходах за 3 квартал,
которые превысили ожидания аналитиков на 15%. Выручка
выросла на 45% год к году. Компания также объявила о
планах расширения операций в три новые страны."
```

### Шаг 2: Спросить LLM

Мы отправляем это в LLM с промптом:

```
"Пожалуйста, проанализируй эту новость и оцени риски
для CryptoMine Inc. по шкале от 1 до 10."
```

### Шаг 3: Получить оценку риска

```
Ответ LLM:

Рыночный риск: 5/10
- Криптовалютный рынок волатилен
- Но фундаментальные показатели компании сильные

Кредитный риск: 2/10
- Сильная прибыль говорит о здоровых финансах
- Планы расширения указывают на уверенность

Риск ликвидности: 3/10
- Акции майнеров могут быть неликвидны
- Но это крупная компания

Операционный риск: 4/10
- Расширение несёт риск исполнения
- Текущие операции выглядят стабильно

Регуляторный риск: 6/10
- Регулирование крипты неопределённо
- Международное расширение добавляет сложности

Сентимент-риск: 3/10
- Положительные новости улучшат настроения
- Превышение прогноза по прибыли благоприятно

Общая оценка: 3.8/10 (Низкий риск)
Направление: Снижается (становится безопаснее)
```

### Шаг 4: Принять решение

```
Общая оценка: 3.8 (Низкий риск)
→ Сигнал: ПОКУПАТЬ
→ Размер позиции: 80% от максимума
→ Причина: Хорошая прибыль, разумная оценка
```

## Почему использовать LLM для этого?

### Скорость
- Человек: 3 часа на анализ одной компании
- LLM: 30 секунд на анализ одной компании
- Можно анализировать 100+ компаний ежедневно

### Последовательность
- Люди устают и делают ошибки
- LLM применяет одинаковые критерии каждый раз
- Нет эмоциональной предвзятости

### Охват
- Люди не могут прочитать всё
- LLM может обработать тысячи статей
- Ничего важного не пропустится

### Языки
- LLM могут читать на многих языках
- Доступ к глобальным источникам новостей
- Находить информацию, которую другие могут пропустить

## Ограничения (Важно!)

LLM не идеальны. Вот их слабости:

### 1. Они могут ошибаться
```
LLM: "Риск низкий!"
Реальность: Компания банкротится через месяц

Почему? LLM знает только публичную информацию.
Она не может видеть секретные проблемы.
```

### 2. Они не хрустальные шары
```
Низкий риск ≠ Гарантированная прибыль
Высокий риск ≠ Гарантированный убыток

Оценка риска — это о вероятности,
не о определённости.
```

### 3. Устаревшая информация
```
Знания LLM имеют дату отсечки.
Очень свежие события могут не быть включены.
Всегда проверяй последние новости отдельно.
```

### 4. Галлюцинации
```
Иногда LLM придумывают факты.
Всегда проверяй важную информацию
из надёжных источников.
```

## Крипта vs. Акции

Этот метод работает для обоих, но с различиями:

| Аспект | Акции | Крипта |
|--------|-------|--------|
| Источники новостей | SEC документы, звонки по прибыли | Twitter, Discord, Telegram |
| Факторы риска | Прибыль, менеджмент | Технологии, регулирование |
| Волатильность | Обычно ниже | Обычно выше |
| Качество данных | Высокое (регулируется) | Смешанное (нерегулируемое) |
| Часы торговли | Только рыночные часы | 24/7 |

## Словарь

| Термин | Простое определение |
|--------|---------------------|
| **LLM** | ИИ, который может читать и понимать текст |
| **Оценка риска** | Число (1-10), показывающее опасность инвестиции |
| **Сентимент** | Общее настроение или чувство о чём-то |
| **Волатильность** | Насколько сильно цены прыгают вверх-вниз |

| **API** | Способ компьютерам общаться друг с другом |
| **Промпт** | Инструкции, которые ты даёшь LLM |

## Частые вопросы

### В: Могу ли я использовать бесплатные LLM вроде ChatGPT для этого?
**О:** Да, для практики и обучения. Для серьёзной торговли тебе могут понадобиться специализированные финансовые LLM или API.

### В: Насколько точна оценка рисков LLM?
**О:** По-разному. Думай об этом как об одном инструменте среди многих. Используй его вместе с другими методами анализа.

### В: Нужно ли мне знать программирование?
**О:** Не для начала. Ты можешь использовать LLM через интерфейсы чата. Программирование поможет для автоматизации позже.

### В: Сколько это стоит?
**О:** Бесплатные версии существуют для практики. Доступ к API стоит по-разному ($0.001 - $0.10 за анализ).

### В: Может это сделать меня богатым?
**О:** В торговле нет гарантий. Оценка рисков LLM — это инструмент, а не волшебная машина для денег.
