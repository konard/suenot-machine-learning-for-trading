# 02 API доступ к рыночным данным

Существует несколько вариантов доступа к рыночным данным через API с использованием Python.

## pandas datareader

Ноутбук [01_pandas_datareader_demo](01_pandas_datareader_demo.ipynb) представляет несколько источников, встроенных в библиотеку pandas.
- Библиотека `pandas` обеспечивает доступ к данным, отображаемым на веб-сайтах, с помощью функции read_html
- Связанная библиотека `pandas-datareader` предоставляет доступ к API эндпоинтам различных поставщиков данных через стандартный интерфейс

## yfinance: Рыночные и фундаментальные данные Yahoo! Finance

Ноутбук [yfinance_demo](02_yfinance_demo.ipynb) показывает, как использовать yfinance для загрузки различных данных из Yahoo! Finance. Библиотека обходит устаревание API исторических данных путем надежного и эффективного скрапинга данных с веб-сайта с помощью pythonic API.

## LOBSTER тиковые данные

Ноутбук [03_lobster_itch_data](03_lobster_itch_data.ipynb) демонстрирует использование данных книги ордеров, предоставляемых LOBSTER (Limit Order Book System - The Efficient Reconstructor), [онлайн](https://lobsterdata.com/info/WhatIsLOBSTER.php) инструмент данных книги лимитных ордеров, который нацелен на предоставление простых в использовании, высококачественных данных книги лимитных ордеров.

С 2013 года LOBSTER выступает в качестве поставщика данных для академического сообщества, предоставляя доступ к реконструированным данным книги лимитных ордеров для всей вселенной акций, торгуемых на NASDAQ. Совсем недавно он начал предлагать коммерческий сервис.

## Quandl

Ноутбук [04_quandl_demo](04_quandl_demo.ipynb) показывает, как Quandl использует очень простой API для предоставления своих бесплатных и премиум данных. См. [документацию](https://www.quandl.com/tools/api) для получения более подробной информации.

## zipline & Quantopian

Ноутбук [zipline_data](05_zipline_data.ipynb) кратко знакомит с библиотекой бэктестинга `zipline`, которую мы будем использовать на протяжении всей этой книги, и показывает, как получать доступ к данным о ценах акций во время выполнения бэктеста. Для установки, пожалуйста, обратитесь к инструкциям [здесь](../../installation).
